# 箱画像 (Hakogazou) - ボードゲーム箱ジェネレーター

2D画像からボードゲームの箱の3Dビジュアライゼーションを生成するWebツールです。箱の各面（前面、背面、側面、上面、底面）の画像をアップロードして、結果の3Dモデルをインタラクティブに確認できます。

## 機能

- **3Dプレビュー**: 箱をリアルタイムで3Dレンダリングします。
- **インタラクティブな操作**: 回転、ズーム、カメラアングルの調整が可能です。
- **カスタマイズ**:
    - 6面すべての画像をアップロード可能。
    - 背景画像の変更。
    - 箱の向き（縦置き/横置き）の調整。
    - 影の表示や自動回転の切り替え。
- **エクスポート**: 現在の表示をPNG画像としてダウンロードできます。

## セットアップ

このプロジェクトは最新のWeb標準を使用しており、ローカルで実行するにはNode.jsが必要です。

1.  **依存関係のインストール**:
    ```bash
    npm install
    ```

## 使い方

1.  **開発サーバーの起動**:
    ```bash
    npm run dev
    ```
    ローカルサーバー（通常は `http://localhost:5173`）が起動し、アプリケーションの表示や編集が可能になります。

2.  **本番用ビルド**:
    ```bash
    npm run build
    ```
    `dist` ディレクトリに本番用のビルドファイルが生成されます。

3.  **ビルドのプレビュー**:
    ```bash
    npm run preview
    ```

## プロジェクト構造

- `index.html`: メインのエントリポイントとUI構造。
- `main.js`: Three.jsを使用した3Dレンダリングのコアロジック。
- `style.css`: アプリケーションのスタイル設定。
- `package.json`: 依存関係とスクリプトの設定。
